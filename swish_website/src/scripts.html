Related Scripts and Projects

AutoSwish
A rewritten (but not widely tested) version of the classic front-end to Swish-e.
GoSwish
GoSwish is a cgi-bin front-end to Swish-e, for use on OS/2 web servers. It uses Swish-e 1.3.1 (as ported by Chris McRae). GoSwish provides a web-based method of
  • specifying and generating Swish-e indices, and
  • searching these indices using a simple html form.
and it does this with out requiring any knowledge of Swish-e (though knowledge of Swish-e will help users optimize their indices).
Java Wrapper for SWISH
Swish.jsp uses Swish-e to dynamically generate XML from the search executed on the indexes. When executing a search, the application will create the XML structure based on the search parameters which are set in the config file. It can be used as a generic search engine for any web site since it is fully configurable.
JSWISHI
A "feature-laden" front-end to Swish-e
Simple Web Search
A PHP3 script by David Norris.
Sman
An enhanced version of apropos that provides free text searching on your system's man pages (using Swish-e), by Josh Rabinowitz.
swish-cgi.pl
John Millard's Perl front-end to Swish-e.
SWISH-Search 0.1
Reenen Kroukamp's Perl front-end to Swish-e.
Swish-e and PHP (in French)
A PHP 4 class that uses Swish-e and returns the results in a table. It takes into account META names and some parameters as well.
SWISH-EX — A COM/ActiveX Component
If you are using Windows Active Server Pages (ASP) you need this!
Swish-eX
A complete package for Windows that includes a web server, a web front-end to Swish-e (Autoswish), converting filters (pdf, doc, rtf), and Swish-e.
SWISHED
Swish-e server code by Josh Rabinowitze. Runs under Apache and mod_perl.
